# results/

This folder contains the output files generated by the **CaloryCal** pipeline.

---

## Structure

```bash
results/
├── images/
│   ├── segmentation_result.jpg
│   └── depth_map.jpg
├── tables/
│   ├── portion_summary.csv
│   └── calorie_estimates.csv
```

---

## images/

Contains visual outputs produced by the pipeline.

### Example files
- `segmentation_result.jpg` — segmentation masks and predicted food labels
- `depth_map.jpg` — depth estimation output for the same input image

These images help demonstrate how the model interprets the food scene before portion and calorie estimation.

---

## tables/

Contains structured outputs produced during analysis.

### Example files
- `portion_summary.csv` — per-food relative portion summary
- `calorie_estimates.csv` — per-food calorie estimates and total calorie output

These files make it easier to inspect and reuse the pipeline outputs outside the notebook.

---

## Notes

- The outputs in this folder are generated from the current prototype pipeline.
- The current system uses **relative portion estimation**, not calibrated real-world measurements.
- Results are intended for demonstration, analysis, and future model improvement.

---

## Summary

This folder provides the final visible evidence of the pipeline, including:
- visual segmentation results
- depth outputs
- relative portion estimates
- calorie estimation tables
